# 🚀 FocusBoard – Productivity Dashboard

FocusBoard is a fully responsive, multi-feature productivity web application built using **Vanilla JavaScript, SCSS, and modern Web APIs**.

It combines task management, time planning, Kanban workflow, Pomodoro timer, sticky notes, weather integration, and theme customization into one clean, intuitive interface.

🔗 **Live Demo:** https://focusboardd.netlify.app/

---

## ✨ Key Features

### 📋 Task Manager
- Add, edit, delete tasks
- Mark as important or completed
- Smart filtering (pending/completed)
- Persistent storage using localStorage

### 📊 Kanban Board
- Drag & drop task movement
- Column-based workflow (To Do / In Progress / Done)
- Dynamic task counters
- State persistence after drop

### 🗒 Sticky Notes
- Masonry-style layout (CSS columns)
- Live editing (title & description)
- Real-time search filtering
- Clean input handling before storage

### ⏳ Pomodoro Timer
- Work / break session toggle
- Pause / reset controls
- Session state persistence

### 📅 Daily Planner
- 24-hour structured time blocking
- Auto-save inputs
- Organized schedule layout

### 🌤 Smart Dashboard
- Geolocation-based weather
- Real-time clock
- Motivational quotes
- Dynamic backgrounds

### 🎨 Theme Engine
- Multiple themes (Light, Dark, Forest, Sunset, Midnight)
- CSS variable-based system
- Theme persistence
